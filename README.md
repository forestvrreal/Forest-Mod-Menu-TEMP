# 🌲 Forest Mod Menu (TEMP)

A clean and powerful Android mod menu built using **BNM**.

---

## 🚀 Features

* Smooth and responsive UI
* Native C++ backend integration
* Modular and easy to expand
* Built with performance in mind
* Clean project structure for quick edits

---

## 🛠 Built With

* **BNM (By Name Modding)**
* C++ (NDK)
* Kotlin / Gradle
* Dobby (hooking framework)

---

## 📂 Project Structure

```
app/
 ├── src/main/
 │   ├── cpp/        # Native code (C++)
 │   ├── java/       # App logic
 │   └── AndroidManifest.xml
 ├── build.gradle.kts
```

---

## ⚙️ Setup

1. Clone the repo:

```
git clone https://github.com/forestvrreal/Forest-Mod-Menu-TEMP.git
```

2. Open in Android Studio

3. Build & run

---

## ⚠️ Notes

* Some dependencies are included as submodules
* Make sure to initialize them if needed:

```
git submodule update --init --recursive
```
